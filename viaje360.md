# Viaje360 - Tu Agente Personal de Viajes

Una aplicación web moderna para planificación de viajes con IA, gamificación y funciones sociales.

## 🌟 Características Principales

### 🧠 Planificación Inteligente
- **IA Personalizada**: Recomendaciones basadas en preferencias individuales
- **Itinerarios Dinámicos**: Ajustes automáticos según clima y disponibilidad
- **Gestión de Presupuesto**: Control en tiempo real de gastos

### 🎮 Gamificación
- **Colección de Monumentos**: Sistema tipo Pokémon GO para lugares icónicos
- **Sistema de Niveles**: Progresión basada en actividades y exploración
- **Trofeos y Logros**: Reconocimientos por diferentes tipos de actividades
- **Quiz Culturales**: Aprende sobre destinos mientras juegas

### 🌍 Exploración Social
- **Compartir Experiencias**: Integración con redes sociales
- **Contenido Optimizado**: Posts automáticos para Instagram, Facebook, Twitter
- **Perfil de Viajero**: Estadísticas personales y logros

### 📱 Funcionalidades Core
- **Búsqueda de Destinos**: Exploración intuitiva de lugares
- **Reservas Integradas**: Vuelos, hoteles y actividades
- **Alertas Meteorológicas**: Recomendaciones adaptativas
- **Navegación Offline**: Mapas y guías sin conexión

## 🚀 Tecnologías Utilizadas

- **Frontend**: React 18 + TypeScript
- **UI Framework**: Shadcn/ui + Tailwind CSS
- **Estado**: Zustand con persistencia
- **Routing**: Single Page Application con navegación por tabs
- **Iconos**: Lucide React
- **Build Tool**: Vite

## 📱 Estructura de la Aplicación

### Páginas Principales
1. **Inicio** - Dashboard personalizado con resumen de viaje actual
2. **Explorar** - Búsqueda de destinos, actividades y quiz
3. **Mi Viaje** - Itinerario detallado y gestión de reservas
4. **Colección** - Monumentos coleccionados, trofeos y logros
5. **Perfil** - Configuración personal y estadísticas

### Componentes Clave
- **Header**: Navegación superior con nivel y puntos
- **Navigation**: Barra inferior con navegación principal
- **Store**: Gestión de estado global con Zustand

## 🎯 Experiencia de Usuario

### Para Nuevos Usuarios
- Onboarding intuitivo sin registro obligatorio
- Demostración inmediata de funcionalidades
- Gamificación desde el primer uso

### Para Usuarios Registrados
- Persistencia de datos y progreso
- Recomendaciones personalizadas

## 🔧 Instalación y Desarrollo

```bash
# Instalar dependencias
pnpm install

# Ejecutar en desarrollo
pnpm run dev

# Construir para producción
pnpm run build

# Verificar código
pnpm run lint
```
